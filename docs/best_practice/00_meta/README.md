# LLM Handbook (expanded, source-driven)

Статус: **structured into full course** (2026-02-21)

## Цель
Собрать из исходных ссылок полноценный локальный учебник по LLM-разработке:
- без шума,
- с проверкой фактов/дат,
- с прикладными best practices,
- в формате, удобном для повторного использования.

## Структура
- `00_meta/` — карта проекта, прогресс, инвентаризация ссылок, программа курса (`course_program.md`)
- `01_foundations/` — модуль 1
- `02_context-engineering/` — модуль 2
- `03_agents-mcp/` — модуль 3
- `04_rag/` — модуль 4
- `05_security/` — модуль 5
- `06_devops/` — модуль 6
- `07_models/` — модуль 7
- `08_playbooks/` — модуль 8
- `09_appendix/` — модуль 9 (глоссарий/референсы)
- `10_capstone.md` — модуль 10 (итоговый проект)
- `sources/` — source notes по первичным ссылкам

## Принцип отбора
1. Сначала официальные docs/blog/курсы.
2. Затем практические статьи/гайдлайны.
3. Репозитории не разбираем глубоко, если это не reference-doc с методологией.

## Как проходить курс (рекомендуемый порядок)
1. `00_meta/course_program.md`
2. `00_meta/study_guide.md` (время, must-pass, навигация)
3. Модули `01` → `08` по порядку
4. `09_appendix` как справочник
5. `10_capstone.md` как итоговая сборка

Каждый модуль содержит `00_module_map.md` с целями, уроками, lab и checkpoint.
