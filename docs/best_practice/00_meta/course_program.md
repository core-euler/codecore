# LLM Handbook — Курс по разработке с LLM (v2)

Статус: 2026-03-21
Формат: 11 модулей, 34 урока, RAG-чат + самостоятельное чтение
Цель: довести разработчика до уровня уверенной production-работы с LLM/agentic стеком.

## Формат обучения

- **Основной**: чат с LLM-учителем (RAG по .md файлам курса)
- **Два режима**: лекции (теория) и семинары (практика)
- **Прогрессия**: модули открываются последовательно
- **Fallback**: full_chapter.md в каждом модуле для самостоятельного чтения
- **Темп**: 4–6 недель (2–3 модуля в неделю) или интенсив 10–14 дней

- Пройдены все 11 модулей последовательно.
- Есть минимум 3 production-ready шаблона (playbooks/skills).
- Есть 1 capstone-проект с документацией и тестами.
- Есть чеклисты по security/cost/quality.
- Есть собственный decision framework по выбору стека.
