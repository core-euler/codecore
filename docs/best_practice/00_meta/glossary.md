# Glossary (v1)

- **Agent loop** — цикл планирования, выполнения, проверки и коррекции действий LLM-агента.
- **Context engineering** — инженерное управление входным контекстом, памятью и ограничениями.
- **Context collapse** — деградация качества из-за чрезмерного сжатия/перезаписи контекста.
- **MCP** — протокол подключения моделей к инструментам/данным.
- **RAG** — retrieval-augmented generation: ответы на базе внешнего извлечения данных.
- **Guardrail** — ограничитель поведения (policy, validation, filters, HITL).
- **Spec-first** — сначала формализация требований, затем реализация.
- **Playbook** — воспроизводимый процесс выполнения типовой задачи.
